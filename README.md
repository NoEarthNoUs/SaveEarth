친환경에 관심은 있지만 어떤 식으로 실천해야 하는지 모르는 분들이 많습니다.

Save Earth는 친환경에 관심이 있는 모든 분들에게 지구를 지킬 수 있는 다양한 정보를 제공하고자 합니다!

Save Earth와 함께 지구를 지켜요! 🙌🌏💪

## 👨‍👩‍👧‍👦 사용자

- 환경에 관심이 있지만 정보를 찾기 어려워 하는 사람
- 제로 웨이스트 라이프에 관심이 있는 사람
- 채식주의자
